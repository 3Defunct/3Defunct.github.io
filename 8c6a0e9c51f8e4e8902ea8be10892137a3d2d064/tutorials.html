Autodesk 3DSMax Tutorials

Notes

Viewport layout tab in the bottom left.
Create object with the right side menu plus icon.
Very similar to Maya overall.

Notes

To change a viewport camera, use the associated letter with viewport selected, e.g. T for Top, P for Perspective.
Another way to change camera type is to click the current view type, which shows a list of other options.
Viewport layout tab in the bottom left.
Right click the grey area between viewports to get reset layout option.
Alt+W to maximise viewport.
Shading viewport option next to current viewtype option, can use this to get a higher quality version/wireframe etc.
F3 to quikcly switch between wireframe and shading view. F4 to have both views at once.
Middle mouse button to pan the view.
Alt + Middle Mouse Button to orbit view.
Scroll wheel to zoom.
Ctrl + Alt + Middle Mouse Button + Mouse up/down for a slower zoom.
Z centers object in viewport, similar to Maya's F.
More camera navigation options in the bottom right.

Notes

Create objects from the right side menu.
Objects have different orintation dependant on the viewport you create them in.
Use autogrid option (above the objects in the object type menu) to orient your new object along the axis of a pre-existing one.
On the modify object menu (next to the create object on the right), you can click the color box next to the object name and change the wireframe color.

Notes

Right click value spinners to set value to 0/lowest, good for zeoing out location.
Saving is standard, no different file types like in maya.
X,Y,Z coords of object at the bottom of the screen.

Notes

Select object button is square with the mouse over it in the top bar menu on the left.
Hold Ctrl to select multiple objects at a time, and Alt to deselect.
Window/Crossing (grey dashed box with blue cube) button on the topbar can be used to change slection mode, between all objects in rectangle and all objects entirely in recetangle.
Selection sets (Middle of top bar) are a useful way of grouping objects just for selection.

Notes

Mostly the same as Maya.
Angle snap button is corner with angle (lol) with magnet, right click to change angle amount. Shortcut is A.
Hold down left click on scale tool button and select bottom option to get a squash and stretch scale.
Bottom of the screen XYZ values change depending on what tool is in use.
Recitile icon next to XYZ values can be clicked to add values to movement/scale etc without having to do the math yourself.
Align tool is two bar graph looking thingy to the right of the create selection set object in the top toolbar. Works by selecting object to move then align to object.

Notes

Heirarchy command panel next to the object modify panel. Affect pivot only is the first option. You will then be able to move the pivot, and there are other pivot options in this menu.

Notes

Cooridnate system option is to the right of the scale tool, and determines which way the X,Y,Z axis face. Boxes at the bottom will always use world in absolute mode. Local is based on object orintation, useful for rotation.

Notes

Hold Shift and move/rotate/scale and object to clone it, and a dialogue box appears.
In dialogue can specify multiple duplicates spaces evenly apart
Different copy types: Copy is a copy, completely different object. Instance means they're the same object, changing one effects the other in modifiers, saves having to do minute changes multiple times.

Notes

To add a modifier just go to the modifier tab and select one from the dropdown tab under the name.
Similar to the maya modifiers but seemingly much better. Edit Poly seems to be like regular vertex editing in Maya.
Taper: Flare one or both sides in or out
Bend: Bend the object
Edit Poly: Edit individual vertices/edges/faces as you would in Maya.
Shell: Turn a regular faced object into a double sides one, like an empty box or pipe.
Right click a modifier and click Collapse All to set all of these modifiers to permanent.
Use mirror tool (Directly to the right of create selection set) to either mirror an object on its axis or duplicate it.
Use merge option to import one scenes specific objects into another.

Notes

Can convert to poly with: Modelling -> Polygon Modeling -> convert to poly.
Or right click -> convert to -> Convert to editable poly
Or right click -> editable poly in modifier stack
1 for vertex, 2 for edge, 3 for faces, 4 for the prior three, 5 for object
Modeling tool window can take a second to appear
Ctrl + Shift to duplicate drag when in editable poly mode, different from what's in the video.
5 to switch between object and poly edit mode
Clone to element keeps two polygons as the same object.
Loop and ring in the modify selection window help you select edge loops and rings.
Select edge then shift click on next edge to get ring of edges, double click on one edge for edge loop.
Connect button connects all selected edges with an additional edge. CConnect settings also lets you add more edge loops in extra settings.
Constrains in the modelling edit menu can be used so a vertex can only be moved along an edge/face/normal, super useful.
Has a weild tool, weilds together all selected vertices.
Target weild tool works similarly to the one you're used to in Maya, just click and drag one verticie onto another.
Swift loop is insert edge loop from Maya almost exactly.
Polygons should almost always have four sides, have smooth edge loops in grid pattern.

Notes

NURMS time. Smooths things out similar to Mayas smooth preview.
Ctrl + double click a neighbouring face to select a face loop.
Hold shift on switf loop tool to snap loop to center.

Notes

Double click one edge of loop to select whole loop.
Loop cylinder edge option hidden in loop settings.
Group objects by clicking group then group from the top label menu.
