Notes

To change a viewport camera, use the associated letter with viewport selected, e.g. T for Top, P for Perspective.
Another way to change camera type is to click the current view type, which shows a list of other options.
Viewport layout tab in the bottom left.
Right click the grey area between viewports to get reset layout option.
Alt+W to maximise viewport.
Shading viewport option next to current viewtype option, can use this to get a higher quality version/wireframe etc.
F3 to uqikcly switch between wireframe and shading view. F4 to have both views at once.
Middle mouse button to pan the view.
Alt + Middle Mouse Button to orbit view.
Scroll wheel to zoom.
Ctrl + Alt + Middle Mouse Button + Mouse up/down for a slower zoom.
Z centers object in viewport, similar to Maya's F.
More camera navigation options in the bottom right.
